# Cricket World Cup-2019
An android app for the upcoming ICC Cricket World Cup-2019.

## Features
- Full Schedule
- Live Score
- Live TV
- Team Squad
- Player Details
- Ranking
- Points Table and Many other features. 

## Technology Used
- Android
- Kotlin
- XML
- Retrofit2
- CricAPI
- YoutubeStreaming API
